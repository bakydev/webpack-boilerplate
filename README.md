# Commands

## Choose the boilerplate

### For `Vanilla JavaScript`

```javascript
branch javascript
```

### For `React`

```javascript
branch react
```

## Installation

Use node package manager [npm](https://https://www.npmjs.com/) to install the project.

```bash
npm install
```

## Usage

Development:

```javascript
npm run dev
```

Production:

```javascript
npm run build
```
